From this directory, you should be able to compile and run the program using the following bash commands:

mkdir bin
javac src/edu/lewisu/cpsc50100/*.java -d bin
cd bin
java edu.lewisu.cpsc50100.Assignment04 ../Scores0.txt

Additional input files are available in the same format.

java edu.lewisu.cpsc50100.Assignment04 ../Scores1.txt
java edu.lewisu.cpsc50100.Assignment04 ../Scores2.txt
java edu.lewisu.cpsc50100.Assignment04 ../Scores3.txt
